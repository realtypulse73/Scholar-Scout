# ScholarScout Roadmap

## Phase 1: MVP

- Launch a usable scholarship matching product.
- Stand up the NestJS API and PostgreSQL data model.
- Support users, student profiles, programs, and baseline matching.

## Phase 2: Intelligence Layer

- Add richer ranking signals and explainability.
- Introduce the dedicated matching and ML services.
- Build a feature store and acceptance prediction model.
- Add recommendation scoring APIs, bandit routing, uplift models, and a causal inference layer.
- Track recommendation quality and feedback loops.

## Phase 3: Marketplace + Monetization

- Add program sponsorships, premium distribution, and billing.
- Enable marketplace workflows for schools and providers.
- Integrate Stripe subscriptions and webhook processing.
- Build school accounts, program upload pipelines, and school analytics.

## Phase 4: AI + Automation

- Introduce copilots, workflow automation, and smart messaging.
- Add async orchestration through queue-backed jobs.
- Add an AI chat assistant and planning agent for guided workflows.

## Phase 5: Governance + Enterprise

- Build auditability, compliance controls, and enterprise admin tools.
- Support approval workflows, policy simulation, and reporting.
- Extend the simulation engine for policy, pricing, and planning analysis.
